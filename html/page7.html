# HTML 段落

HTML 可以将文档分割为若干段落。

# HTML 段落

段落是通过 <p> 标签定义的。

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>页面标题</title>
</head>
<body>

<p>这是一个段落。</p>
<p>这是一个段落。</p>
<p>这是一个段落。</p>

</body>
</html>

提示

注意：浏览器会自动地在段落的前后添加空行。（</p> 是块级元素）

# 不要忘记结束标签

即使忘了使用结束标签，大多数浏览器也会正确地将 HTML 显示出来：

<!DOCTYPE html>
<html>
<head> 
<meta charset="utf-8"> 
<title>页面标题</title> 
</head>
<body>

<p>这是一个段落
<p>这是另一个段落

</body>
</html>

上面的例子在大多数浏览器中都没问题，但不要依赖这种做法。忘记使用结束标签会产生意想不到的结果和错误。

提示

在未来的 HTML 版本中，不允许省略结束标签。

# HTML 折行

如果您希望在不产生一个新段落的情况下进行换行（新行），请使用 <br /> 标签：

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>页面标题</title>
</head>
<body>

<p>这个<br>段落<br>演示了分行的效果</p>

</body>
</html>

<br /> 元素是一个空的 HTML 元素。由于关闭标签没有任何意义，因此它没有结束标签。
